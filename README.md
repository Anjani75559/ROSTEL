# 🏨 ROSTEL - Hostel Management System

Rostel is a complete web-based hostel management system built with HTML, CSS, and JavaScript. It features a dual-login system for both hostel owners and students. Owners can manage rooms, allocate beds, and oversee student admissions through a dedicated admin dashboard. Students can register via an online admission form, select their room, upload ID proof and photo, and access their personal student portal. Powered by Supabase as the backend database for storing and managing all student and hostel data in real time.

## ✨ Features
- 🔐 Dual login system (Owner & Student)
- 📋 Online student admission form
- 🛏️ Room selection and allocation
- 📸 ID proof and photo upload
- 👤 Student personal portal
- 🏠 Owner admin dashboard

## 🛠️ Tech Stack
- HTML, CSS, JavaScript
- Supabase (Database)
- GitHub Pages (Hosting)
